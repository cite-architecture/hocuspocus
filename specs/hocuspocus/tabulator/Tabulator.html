Tabulator

You can convert archival text files in a corpus to a tabular delimited-text format that is equivalent under the OHCO2 model of citable texts. It is possible to specify by URN a single document to convert, or to convert the entire corpus documented in the Text Inventory.

Example

Using the files in this root directory and this TextInventory file, if we request a tabular representation of urn:cts:greekLit:tlg0012.tlg001.msA, the results will be in a file named Iliad-A.txt, and will have 33 lines.

If we tabulate all files in the same corpus in this directory, we will generate 2 files.
