Package: edu.holycross.shot.hocuspocus

[Groovy] Class Tabulator

    • edu.holycross.shot.hocuspocus.Tabulator

    • Tabulator is a utility class for working with canonically citable texts in the OHCO2 model, and converting representations in XML files to a tabular format.

It includes by default a mapping of the TEI namespace to the prefix abbreviation 'tei' : other mappings can be added with the addNSMapping method().
